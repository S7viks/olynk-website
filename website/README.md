## üéØ Quick Start

### For Designers
1. Read: `WEBSITE_SPECIFICATION.md` sections:
   - Design Philosophy
   - Brand Identity
   - Homepage Structure
2. Create wireframes based on structure
3. Design high-fidelity mockups
4. Reference `COPY_MASTER.md` for actual copy

### For Developers

**Using Lovable.dev?** üöÄ
1. Read `LOVABLE_INSTRUCTIONS.md` (5-minute read)
2. Open `LOVABLE_PROMPT.md`
3. Copy the entire prompt
4. Paste into Lovable.dev
5. Let it generate the complete website
6. Done! (Iterate from there)

**Building Manually?** üõ†Ô∏è
1. Read: `BUILD_PROMPT.md` - Complete build instructions
2. Read: `WEBSITE_SPECIFICATION.md` sections:
   - Technical Stack
   - Homepage Structure (for components)
   - Implementation Roadmap
3. Setup Next.js project with specified stack
4. Implement components section by section
5. Pull copy from `COPY_MASTER.md`

**BUILD_PROMPT.md contains:**
- Complete technical requirements
- Every section detailed with exact specifications
- Design system (colors, typography, spacing)
- Responsive design requirements
- Animation specifications
- Asset requirements
- Testing checklist
- Deployment guide

1. Read: `COPY_MASTER.md` completely
